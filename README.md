### Dependencies

*   Node: v8.9.4
*   Yarn: v1.3.2

### Setup Instructions

1.  Install dependencies
2.  Run `yarn` _or_ `yarn install`

### Common Commands

*   Start Node server: `yarn start`
*   Run Jest test suite: `yarn test`
*   Build the project: `yarn build`
